Using Abstract Syntax Trees for Text to codeGenerationPrajjwal Bhattaraipb2276@nyu.eduAdvised by: Riyadh BaghdadiABSTRACTTransformer based, pre-trained models have been achievingstate-of-the-art results in a lot of translation and generationtasks. For the text-to-code generation tasks, the state-of- the-art models are either pre-trained or fine-tuned on one specificprogramming language. Training a model on an abstractsyntax tree (AST) helps generalize this problem, not onlymaking it language agnostic, but extracting and integratingsyntactical features inherent to any programming languageand making models less reliant on auxiliary features likeidentifier names. We look into and reproduce two previousmodels that have approached the code generation problemthrough the means of Abstract Syntax Trees, one using RNNbased architecture, the other using transformers. We aimto extend them to datasets that the original work was nottrained in with an overall aim to challenge the performanceof larger state-of-the-art models that do not use AbstractSyntax Trees.Reference Format:Prajjwal Bhattarai. 2021.
